# SYNERA Frontend Showcase

SYNERA is a multi-organ clinical AI product concept for early risk detection across ECG/cardiology, liver pathology, chronic viral hepatitis screening, ophthalmology, and multi-organ consilium workflows.

## What this repository contains

This public repository contains the **frontend / product showcase only**:

- landing page and product positioning screens;
- static UI for the clinical dashboard experience;
- frontend flows for ECG, hepatology, ophthalmology, patient profile, reports, and consilium views;
- demo copy and non-production interface states.

The goal of this repository is to show the product direction, user experience, and system concept.

## What is not included

The real ML models, model weights, inference pipelines, private validation assets, and production clinical backend are **not included** in this public repository.

Real model execution is kept local/private and can be demonstrated only:

- after a real commercial/research contract;
- during an approved pilot discussion;
- or in an in-person / private live demo.

This protects model IP, clinical data handling, deployment security, and validation integrity.

## ML scope

SYNERA is designed to connect multiple AI modules:

- ECG/cardiology risk analysis;
- liver fibrosis and liver pathology screening;
- chronic viral hepatitis screening support;
- retinal / ophthalmology image analysis;
- multi-organ consilium that combines heart, liver, eye, and metabolic risk signals.

The public UI may show simulated or demo states. Those screens should not be interpreted as public access to the production ML models.

## Medical disclaimer

SYNERA is currently a research and product demonstration project. It is not a certified medical device and must not be used for diagnosis, treatment decisions, or emergency care without proper clinical validation, regulatory review, and contractual deployment controls.
